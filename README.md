# whatsmyip.dev

Another whatsmyip style website. Check it out at [whatsmyip.dev](https://whatsmyip.dev).

## What's it Built With?

* Zeit's Now (Serverless Platform)
* Milligram (CSS Framework)
* store.js (Browser Storage Wrapper)
